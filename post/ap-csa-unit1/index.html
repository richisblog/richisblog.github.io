AP-CSA-UNIT1

Unit 1 : Primitive Types 初始类型

1.1 编程入门，了解一些注意事项

print 和 println 的区别：

两者均适用于输出内容。

System.out.println("hello, world!");
System.out.print("hello, world!");

println 输出内容，并换行。
print 输出内容，不换行。

注释写法：

注释帮助理解代码内容，不参与代码运行。

// this is a comments.

双斜杆后面随便写。

分号

每个独立的代码完成后需要一个；以保证运行。

1.2 变量 variable

声明

变量必须声明，没有声明的变量使用会报错；
变量可以先声明在赋值，但必须有赋值，没有赋值会报错；
变量只能且必须声明一次，多次声明会报错；
变量不能数字开头，必须使用字母或_开头，否则报错；
变量起名要用meaningful name（驼峰命名法），保证读者知道这是什么变量。

原始类型

int：integers 整数
double：decimal number 浮点数
boolean：布尔值
考试只有这三种，所以不多介绍。

PS：data type 分为 初始类型和引用类型。

Reference Types 引用类型

String 字符串，s 要大写，是引用类型。

1.3 基础语法：数学表达式和赋值语句

数学表达式

运算符 描述 例子
+ 加法 a + b
- 减法 a - b
* 乘法 a * b
/ 除法 a / b
% 取模（取余数） a % b

取模常用于测量倍数，语数，循环，奇偶数。

优先级

乘*、除/、取模% > 加+、减-

除法

整数(int)除整数只会得到整数，如果出现小数也只保留整数部分。
举个例子:

int x = 2;
int y = 3;
int z = -2;
System.out.println(x/y);
System.out.println(y/x);
System.out.println(y/z);
System.out.println(z/y);

输出结果为：

0
1
-1
0

但当除数或被除数为小数时，就可以得到小数。
例如：

int x = 2;
double y = 3; //这里改为了小数 double
int z = -2;
System.out.println(x/y);
System.out.println(y/x);
System.out.println(y/z);
System.out.println(z/y);

0.6666666666666666
1.5
-1.5
-0.6666666666666666

赋值符号 =

单个 = 是赋值符号，即将等号右侧的内容赋值给左侧；
等号两侧数据类型不同会报错；
等号 不代表相等，而是赋值。
